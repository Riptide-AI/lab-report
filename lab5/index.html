Start

В результаті виконання лабораторної роботи №5 ми:
- дізнались про бібліотеку jQuery UI та її особливості
- застосували на практиці набуті знання, додавши до веб-сайту необхідні елементи
- переконались у перевагах даної бібліотеки

Також ми порівняли процес виконання 4-ї лабораторної роботи з процесом виконання 5-ї та прийшли до висновку, що використання бібліотеки jQuery UI значно спрощує процес створення контенту та додавання динамічних елементів на сайті.

Тема та постановка задачі:

Тема:
ВИКОРИСТАННЯ В СЦЕНАРІЯХ JAVASCRIPT БІБЛІОТЕК КРОС-БРАУЗЕРНИХ ІНТЕРФЕЙСІВ МЕТОДІВ DOM. БІБЛІОТЕКА JQUERY UI.
ЗАСОБИ НАЛАГОДЖЕННЯ І ТЕСТУВАННЯ СЦЕНАРІЇВ JAVASCRIPT.

Мета: придбати практичні навички роботи jQuery UI, вміти додавати до сайту динамічні елементи цієї бібіліотеки, плагін галереї UniteGallery, кнопки соціальних мереж, інформери та інше.

В ході виконання лабораторної роботи №5 необхідно:
- ознайомитись з теоретичними відомостями про jQuery UI
- навчитися додавати елементи даної бібліотеки до веб-сайту
- визначити основні переваги бібліотеки jQuery
- зробити висновки по роботі з даною бібліотекою.
