У восьмій лабораторній роботі отримали практичні навички роботи взаємодії WEB-застосувань з СУБД, використання сесій і cookies в сценаріях авторизації, розробки сценаріїв обробки даних з файлів.
Ми ознайомилися з теоретичними відомостями до лабораторної роботи №8 (Реалізація БД в СУБД MySQL, PHP - скрипт для внесення даних до БД через html-форму, PHP - скрипт для внесення даних до БД через html-форму)
Завантажили «XAMPP»
Визначили класи об'єктів, дані які будуть зберігатися у БД
Cтворили інфологічну модель
Нормалізували реляційну базу
Створили даталогічну модель
Реалізували базу даних в СУБД MySQL, використовуючи веб-інтерфейс для роботи із сервером MySQL phpMyAdmin.




Порівняння функціоналу сайту на момент проектування та на даний момент

На даний момент на сайті можна:

  • Створити аккаунт, зайти у власний кабінет за допомогою логіна та пароля
  • Можна переглятути свої дані, що буду виведені за допомогою SQL запросів
  • Можна переглянути слайдер з картинками
  • Можна вибрати бажану мову(ua/ru/eng)
  • Можна ознайомитись з прейскурантом
  • Клієнту повністю доступна навігація по сайту

Що вийшло і не вийшло реалізувати


Повністю реалізовано:

  1. Головна сторінка була повністю реалізована, а саме реалізація навігації, зміна мови, візуальна інформація

  2. Пункт ціна також був повністю реалізований, створено привабливий дизайн прейскуранту

  3. Галерея також була повністю завершення, необхідно лише збільшити кількість фотографій для показу


    4. Частково реалізовано:

    1. Команда була реалізована частково, з візуальної частини даний пункт повністю готовий


      2. Не реалізовано:

      1. Про нас невелика інформативна частина сайду, яка ще не була реалізована
